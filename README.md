## GitHub Pages更新方法

ZIPを解凍し、中身をGitHubリポジトリ直下へ上書きアップロードしてください。

アップロード対象：

- index.html
- styles.css
- app.js
- assets/
- data/
- README.md
- CHANGELOG.md
- VERSION.txt

アップロード後、ブラウザ表示が古い場合は Ctrl + F5 で再読み込みしてください。
