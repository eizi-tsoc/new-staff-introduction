# TSOC 新入職員紹介ツール Web Ver.1.0.0

ローカル版 Ver.1.10.6 をベースにした GitHub Pages 用の静的Webアプリです。

## GitHub Pages への配置

1. GitHubで `tsoc_staff_intro_tool` などのリポジトリを作成します。
2. このフォルダ内のファイルをすべてアップロードします。
3. GitHubの `Settings` → `Pages` を開きます。
4. `Deploy from a branch` を選び、`main` / `/root` を指定します。
5. 数分後に発行されたURLへアクセスします。

## 注意

- 入力した職員データ・写真・マスタはブラウザの localStorage に保存されます。
- サーバーへ自動保存はされません。
- 別PCで使う場合は「データ保存」でJSONを書き出し、「データ読込」で取り込んでください。
- PDFはブラウザの印刷画面から「PDFに保存」を選択してください。

## 機能

- 氏名、フリガナ、所属、職種、入職日、一言、写真の登録
- 写真プレビュー
- 登録済み一覧、ダブルクリック編集
- マスタ管理
- 5名固定レイアウト、6名以上は複数ページ
- PDF作成/印刷
- LINE画像作成
